## 貢獻指南

### 如何參與

1. **報告 Bug**
   - 在 GitHub 上打開 [Issue](https://github.com/Ethanchen168/abyss-rift/issues)
   - 詳細描述問題和重現步驟

2. **建議功能**
   - 描述你想要的新功能
   - 解釋為什麼這個功能有用

3. **提交代碼**
   - Fork 倉庫
   - 創建新分支 (`git checkout -b feature/amazing-feature`)
   - 提交改動 (`git commit -m 'Add amazing feature'`)
   - 推送分支 (`git push origin feature/amazing-feature`)
   - 打開 Pull Request
