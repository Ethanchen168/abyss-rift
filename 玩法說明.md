# 🕹️ 玩法說明

## 控制方式

### 移動系統 ✅

玩家可以在遊戲世界中自由移動。

#### 鍵盤控制

| 按鍵 | 功能 |
|------|------|
| **W** 或 **↑** | 向上移動 |
| **S** 或 **↓** | 向下移動 |
| **A** 或 **←** | 向左移動 |
| **D** 或 **→** | 向右移動 |

#### 鼠標控制

- 點擊屏幕上的位置可移動到該處（計劃中）

## 遊戲界面

### 主界面元素

- **角色**: 屏幕中央的可移動角色
- **地圖**: 遊戲世界地形
- **UI 元素**: 血量、法力、物品欄等 (開發中)

### 視野系統

- 攝像機跟隨角色移動
- 視野範圍涵蓋角色周圍區域
- 探索隱藏區域以發現新內容

## 遊戲提示

### 探索技巧

1. **充分探索**: 不要放過任何角落，可能有隱藏的秘密
2. **閱讀提示**: 注意屏幕上的提示信息
3. **保存進度**: 定期保存遊戲進度 (開發中)

### 完成任務

1. 接受任務
2. 按照任務指示進行
3. 完成目標後返回領取獎勵

### 收集物品

- 在探索中尋找道具
- 物品可能對完成任務有幫助
- 某些物品是隱藏的，需要仔細探索

## 遊戲目標

### 主線目標

1. 進入深淵裂隙
2. 探索未知區域
3. 收集必要物品
4. 解開謎團
5. 完成冒險

### 支線目標

- 發現所有隱藏地點
- 收集所有物品
- 完成特殊任務
- 解鎖成就
