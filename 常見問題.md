# ❓ 常見問題 (FAQ)

## 遊戲相關

### Q1: 這個遊戲是免費的嗎？
**A:** 是的！Abyss Rift 完全免費，並且開源。你可以自由下載和分享。

### Q2: 我可以在手機上玩嗎？
**A:** 目前優化主要針對桌面瀏覽器。手機版本仍在計劃中。

### Q3: 需要網路連接嗎？
**A:** 初次下載需要網路連接，但下載後可以離線遊玩。

### Q4: 遊戲有多長？
**A:** 目前遊戲仍在開發中。完整版本預計在 2026 年底發布。

### Q5: 這是多人遊戲嗎？
**A:** 目前是單人遊戲。多人遊戲功能計劃在未來版本中添加。

### Q6: 我可以自定義角色嗎？
**A:** 目前不支援。自定義系統計劃在 v0.5.0 版本中添加。

## 技術相關

### Q7: 我怎樣才能報告 Bug？
**A:** 
1. 訪問 [GitHub Issues](https://github.com/Ethanchen168/abyss-rift/issues)
2. 點擊 "New issue"
3. 詳細描述問題
4. 包含：操作系統、瀏覽器、重現步驟

**Bug 報告模板**
```
**描述**
簡短描述 bug

**重現步驟**
1. 做什麼
2. 然後做什麼
3. Bug 發生

**預期行為**
應該發生什麼

**實際行為**
實際發生了什麼

**系統信息**
- 操作系統：
- 瀏覽器：
- 版本：
```

### Q8: 我可以建議新功能嗎？
**A:** 絕對可以！
1. 打開 [GitHub Issues](https://github.com/Ethanchen168/abyss-rift/issues)
2. 選擇 "Feature request"

### Q9: 我如何參與開發？
**A:** 
1. Fork 倉庫
2. 創建功能分支
3. 提交 Pull Request
4. 詳細描述你的改動

詳見 [[開發進度|開發進度]] 頁面的貢獻指南。

### Q10: 遊戲用什麼技術開發？
**A:**

- **渲染**: Canvas API

## 玩法相關

### Q11: 我應該如何開始？
**A:**
1. 按照 [[安裝指南|安裝指南]] 安裝遊戲
2. 閱讀 [[玩法說明|玩法說明]] 學習控制
3. 打開 index.html 開始遊玩
4. 使用 WASD 或方向鍵移動

### Q12: 如何移動角色？
**A:**
| 按鍵 | 功能 |
|------|------|
| W 或 ↑ | 向上 |
| S 或 ↓ | 向下 |
| A 或 ← | 向左 |
| D 或 → | 向右 |

### Q13: 遊戲有保存功能嗎？
**A:** 目前沒有。存檔系統計劃在 v0.5.0 版本中添加。

### Q14: 我卡住了怎麼辦？
**A:** 
- 試著探索其他區域
- 重新閱讀遊戲提示
- 在 GitHub 上提問

### Q15: 遊戲有難度設置嗎？
**A:** 目前沒有。難度設置計劃在未來版本中添加。

## 性能相關

### Q16: 遊戲很慢怎麼辦？
**A:**
1. 關閉其他應用程序
2. 清除瀏覽器緩存
3. 更新顯卡驅動程序
4. 嘗試其他瀏覽器

### Q17: 為什麼控制有延遲？
**A:**
1. 檢查 CPU 使用率
2. 關閉後台應用程序
3. 嘗試降低系統負載

### Q18: 我可以改進遊戲性能嗎？
**A:** 如果你有優化建議：
1. 打開 GitHub Issue
2. 詳細描述性能問題
3. 包含系統配置信息
