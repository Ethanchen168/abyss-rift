# 📥 安裝指南

## 系統需求

### 最低要求

- **作業系統**: Windows / macOS / Linux
- **瀏覽器**: 
  - Chrome 90+
  - Firefox 88+
  - Safari 14+
  - Edge 90+
- **RAM**: 512MB
- **磁盤空間**: 10MB

### 推薦配置

- **作業系統**: Windows 10+ / macOS 10.15+ / Ubuntu 20.04+
- **瀏覽器**: 最新版本 Chrome 或 Firefox
- **RAM**: 2GB 或以上
- **網路**: 最初下載時需要（之後離線可玩）

## 安裝步驟

### 方法 1: GitHub Desktop (推薦新手)

1. **下載 GitHub Desktop**
   - 訪問 [GitHub Desktop 官方網站](https://desktop.github.com/)
   - 下載並安裝

2. **克隆倉庫**
   - 打開 GitHub Desktop
   - 選擇 File → Clone Repository
   - 輸入: `Ethanchen168/abyss-rift`
   - 選擇本地存儲位置

3. **打開遊戲**
   - 在文件夾中找到 `index.html`
   - 用瀏覽器打開

### 方法 2: 命令行 (適合進階用戶)

1. **打開終端/命令提示符**

2. **克隆倉庫**
   ```bash
   git clone https://github.com/Ethanchen168/abyss-rift.git
   cd abyss-rift
   ```

3. **打開遊戲**
   - Linux/macOS: `open index.html`
   - Windows: `start index.html`
   - 或直接用瀏覽器打開 `index.html`

### 方法 3: 直接下載

1. **訪問 GitHub 倉庫**
   - 打開 https://github.com/Ethanchen168/abyss-rift

2. **下載代碼**
   - 點擊綠色 Code 按鈕
   - 選擇 Download ZIP

3. **提取文件**
   - 解壓下載的 ZIP 文件

4. **打開遊戲**
   - 用瀏覽器打開 `index.html`

## 首次運行

1. **打開 index.html**
   - 用任何現代瀏覽器打開
   - 遊戲應該立即加載

2. **檢查遊戲**
   - 確認角色出現在屏幕上
   - 嘗試用 W/A/S/D 或方向鍵移動

3. **游玩遊戲**
   - 按照 [[玩法說明|玩法說明]] 開始探索

## 常見問題

### 遊戲無法加載？

- ✅ 確保使用現代瀏覽器
- ✅ 清除瀏覽器緩存 (Ctrl+Shift+Delete)
- ✅ 嘗試用不同瀏覽器
- ✅ 檢查文件路徑是否正確

### 遊戲很慢？

- ✅ 關閉其他應用程序
- ✅ 更新顯卡驅動程序
- ✅ 檢查 GPU 加速是否啟用

### 控制不響應？

- ✅ 點擊遊戲窗口確保獲得焦點
- ✅ 嘗試刷新頁面 (F5)
- ✅ 更換輸入設備

## 更新遊戲

### GitHub Desktop 用戶

1. 打開 GitHub Desktop
2. 在 Current Repository 中選擇 abyss-rift
3. 點擊 "Fetch origin" 然後 "Pull origin"

### 命令行用戶

```bash
cd abyss-rift
git pull origin main
```

## 卸載遊戲

- 簡單地刪除 abyss-rift 文件夾即可
